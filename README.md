# Day 4 Tool Calling Agent

This project implements a manual tool-calling AI agent using LangChain and Ollama.

## Features
- Web search using Tavily
- Text summarization
- Notes generation
- Manual agent loop with JSON tool calls

## Tech Stack
- Python
- LangChain
- Ollama (llama3.2:3b)

## How to Run
```bash
python day4_tool_calling.py
